IGCSE Computer Science – Section 4.1

Types of Software, Operating Systems & Interrupts

Teacher Pane (Concept Mastery & Teaching Flow)

4.1 Types of Software

Key idea: Hardware is useless without software. Software provides instructions that tell hardware what to do.

Software Classification

  • System Software – Controls and manages hardware, provides a platform
  • Application Software – Performs specific user tasks

System Software – Teaching Points

  • Controls hardware resources
  • Provides Human Computer Interface (HCI)
  • Includes OS, utilities, drivers, translators

Application Software – Teaching Points

  • Single programs or suites
  • Executed when required by the user
  • Relies on system software to run

Application Software Examples – Emphasis

  • Spreadsheet – modelling, formulas, graphs
  • Database – tables, records, fields, queries
  • Word Processor – text manipulation
  • Graphics, Photo & Video Editing – media manipulation
  • Control & Measurement – sensors and actuators
  • Apps – mobile‑based applications

System Software Examples – Emphasis

  • Operating System – resource manager
  • Utilities – maintenance tools
  • Device Drivers – hardware communication
  • Compilers & Linkers – program translation

Operating System – Core Functions

  • HCI (CLI vs GUI)
  • Memory management
  • File management
  • Security management
  • Peripheral management
  • Multitasking
  • User account management
  • Interrupt handling
Exam Tip: Students must describe function + purpose. One‑line answers lose marks.

Interrupts – Teaching Flow

  • Interrupt = signal to CPU
  • CPU saves current state
  • ISR executed
  • Task resumed
Analogy: Phone call interrupting your work → you answer → continue work.

Buffers & Interrupts

  • Buffers store data temporarily
  • Used when device speed ≠ CPU speed
  • Improves efficiency and multitasking

Student Pane (Detailed Notes & Heavy Practice)

What is Software?

Software is a collection of programs that instruct computer hardware to perform tasks. Without software, hardware cannot function.

Types of Software

System Software: Controls the computer and manages hardware.
Application Software: Helps users perform tasks.

Application Software – Explained

  • Spreadsheet: Uses rows, columns, cells, formulas, graphs
  • Database: Stores data in tables (records & fields)
  • Word Processor: Create and edit text documents
  • Graphics Software: Bitmap (pixels) & Vector (lines)
  • Photo Editing: Modify images, remove defects
  • Video Editing: Cut, merge, add effects
  • Control Software: Uses sensors & actuators
  • Apps: Mobile applications downloaded from app stores

System Software – Explained

  • Operating System: Manages hardware and software
  • Utilities: Antivirus, backup, compression
  • Device Drivers: Allow hardware to work
  • Compiler: HLL → machine code
  • Linker: Combines object files

Operating System Interfaces

CLI: Text commands, fast, powerful, hard to learn
GUI: Icons, mouse/touch, user‑friendly, memory‑intensive

Interrupts

An interrupt is a signal that causes the CPU to temporarily stop its current task and handle an urgent event.

  • Hardware interrupts
  • Software interrupts
  • User interrupts

Buffers

Buffers temporarily store data to handle speed differences between devices.
