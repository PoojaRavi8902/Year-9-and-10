Lesson 16 – REPEAT UNTIL Loops

Lesson Objectives

  • Understand the structure of REPEAT UNTIL loops
  • Learn how REPEAT UNTIL loops differ from WHILE loops
  • Write algorithms using REPEAT UNTIL loops

What is a REPEAT UNTIL Loop?

A REPEAT UNTIL loop is a type of condition-controlled loop. The loop continues to execute instructions until a specified condition becomes true.

Unlike a WHILE loop, the condition is checked after the instructions are executed. This means the loop will always execute at least once.

General Structure

REPEAT
    statements
UNTIL condition

The instructions inside the loop are executed first. After that, the condition is evaluated. If the condition is false, the loop continues. If the condition is true, the loop stops.

Example 1 – Counting Numbers

i ← 1

REPEAT
    OUTPUT i
    i ← i + 1
UNTIL i > 5

This algorithm displays the numbers from 1 to 5.

Example 2 – Input Validation

A REPEAT UNTIL loop is commonly used to ensure that valid data is entered. 

REPEAT
    INPUT number
UNTIL number >= 0

The loop continues asking for input until the user enters a valid number.

Difference Between WHILE and REPEAT UNTIL

WHILE Loop REPEAT UNTIL Loop
Condition checked before the loop runs Condition checked after the loop runs
The loop may not execute at all The loop always executes at least once
Loop continues while condition is TRUE Loop continues until condition becomes TRUE

Advantages of REPEAT UNTIL Loops

  • Ensures the loop runs at least once
  • Useful for input validation
  • Commonly used in interactive programs

Practice Questions

  1. What is a REPEAT UNTIL loop?
  2. Explain one difference between WHILE and REPEAT UNTIL loops.
  3. Write pseudocode that asks the user to enter a number until the number is positive.

Exam Style Question
