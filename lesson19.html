Lesson 19 – Constants

Lesson Objectives

  • Understand the meaning of constants in programming
  • Identify the difference between variables and constants
  • Use constants in algorithms

What is a Constant?

A constant is a value that does not change during the execution of a program. Once a constant is defined, its value remains fixed.

Constants are often used to represent important values that should remain the same throughout the program.

Examples of Constants

Some common examples of constants include:

  • The value of π (3.142)
  • The number of days in a week (7)
  • The number of months in a year (12)

Using Constants in Algorithms

Constants can be used in calculations where a value must remain unchanged.

Example:

PI ← 3.142

INPUT radius

area ← PI * radius * radius

OUTPUT area

In this example, the value of PI remains constant.

Difference Between Variables and Constants

Variable Constant
Value can change Value cannot change
Used for storing changing data Used for fixed values
Example: totalMarks Example: PI

Advantages of Using Constants

  • Improves readability of programs
  • Prevents accidental modification of important values
  • Makes programs easier to maintain

Example Algorithm

PASS_MARK ← 50

INPUT mark

IF mark >= PASS_MARK THEN
    OUTPUT "Pass"
ELSE
    OUTPUT "Fail"
ENDIF

In this algorithm, the pass mark remains constant throughout the program.

Practice Questions

  1. Define the term constant.
  2. Explain one difference between variables and constants.
  3. Give one example of a constant used in programming.

Exam Style Question
