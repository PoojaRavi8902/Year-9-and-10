Lesson 25 – Total Algorithms

Lesson Objectives

  • Understand the purpose of total algorithms
  • Learn how to calculate totals using loops
  • Apply accumulation techniques in algorithms

What is a Total Algorithm?

A total algorithm is used to calculate the sum of a set of numbers. It works by repeatedly adding values together until all numbers have been processed.

Total algorithms are commonly used when working with arrays or lists of data.

Basic Structure of a Total Algorithm

total ← 0

FOR i ← 1 TO n
    total ← total + value
NEXT i

The variable total accumulates the sum of values as the loop progresses.

Example Problem

Calculate the total marks of five students stored in an array. 

total ← 0

FOR i ← 1 TO 5
    total ← total + marks[i]
NEXT i

OUTPUT total

Example Dataset

Marks: 70  80  65  90  75

The total will be calculated as: 

70 + 80 + 65 + 90 + 75 = 380

How the Algorithm Works

  1. Initialize the total variable to 0.
  2. Read each value in the array.
  3. Add the value to the total.
  4. Repeat until all values are processed.

Combining Total and Average

Total algorithms are often used together with average calculations. 

total ← 0

FOR i ← 1 TO 5
    total ← total + marks[i]
NEXT i

average ← total / 5

OUTPUT average

Why Total Algorithms Are Important

  • Used to process large datasets
  • Help calculate sums and averages
  • Commonly used in real-world programs
