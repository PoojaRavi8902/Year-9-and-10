Lesson 5 – Algorithms

Lesson Objectives

  • Understand the meaning of an algorithm
  • Learn the characteristics of a good algorithm
  • Recognise that algorithms are used in everyday life
  • Understand how algorithms help solve computational problems

What is an Algorithm?

An algorithm is a step-by-step set of instructions used to solve a problem or perform a task. In computer science, algorithms describe the logical sequence of steps that a computer program follows in order to produce a desired output.

Algorithms must be clear, logical, and unambiguous so that both humans and computers can understand them.

Algorithms in Everyday Life

Algorithms are not limited to computer programs. They are used in many everyday activities.

For example, consider the algorithm for making a cup of tea:

1. Boil water
2. Place a tea bag in a cup
3. Pour hot water into the cup
4. Allow the tea to brew
5. Add milk or sugar if required
6. Stir the tea

Each step must be performed in the correct order to achieve the desired result.

Characteristics of a Good Algorithm

A good algorithm should have the following characteristics:

  • Clear and precise – each step must be easy to understand
  • Finite – the algorithm must eventually stop
  • Logical sequence – the steps must follow a sensible order
  • Efficient – it should solve the problem without unnecessary steps

Example Algorithm

Consider an algorithm that calculates the area of a rectangle. 

INPUT length
INPUT width

area ← length × width

OUTPUT area

The algorithm clearly shows the inputs, the processing step, and the final output.

Why Algorithms are Important

  • They provide a clear plan for solving problems
  • They help programmers organise logical steps
  • They reduce programming errors
  • They make programs easier to design and understand

Algorithms Before Programming

Programmers usually design algorithms before writing the actual program code. This helps ensure that the logic of the solution is correct before implementing it in a programming language.

Practice Questions

  1. Define the term algorithm.
  2. Give one example of an algorithm used in everyday life.
  3. List two characteristics of a good algorithm.

Exam Style Question

Explain why algorithms are important when designing computer programs.
