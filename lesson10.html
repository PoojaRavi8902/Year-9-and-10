Lesson 10 – Sequence

Lesson Objectives

  • Understand the concept of sequence in algorithms
  • Recognise how instructions are executed step by step
  • Design simple algorithms using sequential steps

What is Sequence?

Sequence is the simplest control structure used in algorithms and programming. In a sequence structure, instructions are executed one after another in the order they appear.

Each step is performed exactly once, and the program flows directly from one instruction to the next.

Example of Sequence

Consider an algorithm that calculates the area of a rectangle. 

INPUT length
INPUT width

area ← length * width

OUTPUT area

The instructions are executed in the order they are written:

  1. The program receives the input values.
  2. The program calculates the area.
  3. The program displays the result.

Sequence in Everyday Life

Many everyday activities follow a sequence of steps. For example, the process of brushing your teeth:

  1. Pick up the toothbrush
  2. Apply toothpaste
  3. Brush your teeth
  4. Rinse your mouth

These steps must be followed in order to complete the task properly.

Another Example Algorithm

Algorithm to calculate the average of two numbers. 

INPUT number1
INPUT number2

sum ← number1 + number2
average ← sum / 2

OUTPUT average

Each instruction is executed in a fixed order.

Advantages of Sequence Structure

  • Simple and easy to understand
  • Provides a clear logical flow of instructions
  • Used in almost every algorithm

Practice Questions

  1. Define the term sequence.
  2. Explain why sequence is important in algorithms.
  3. Write an algorithm that calculates the perimeter of a rectangle.

Exam Style Question

Write pseudocode to input three numbers and display their total.
