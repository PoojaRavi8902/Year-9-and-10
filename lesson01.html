Lesson 1 – Introduction to Problem Solving

Lesson Objectives

  • Understand what problem solving means in computing
  • Learn steps used to solve computational problems
  • Understand the role of algorithms

Introduction

Problem solving is a fundamental skill in computer science. Before writing a program, the problem must be clearly understood and a logical solution designed.

Steps in Computational Problem Solving

  1. Understand the problem
  2. Identify inputs
  3. Identify outputs
  4. Design the algorithm
  5. Test the solution

Example

INPUT length
INPUT width
area ← length × width
OUTPUT area
