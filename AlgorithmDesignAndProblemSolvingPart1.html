🔄 Program Development Life Cycle (PDLC)

The Program Development Life Cycle describes the stages involved in developing a program from idea to solution.
  • Analysis
  • Design
  • Coding
  • Testing
  • Maintenance
You must describe each stage — not just list them.

📊 Analysis

This stage produces a requirements specification.

Abstraction

Removes unnecessary detail and keeps essential information.

Decomposition

Breaks a problem into smaller parts.

Abstraction = remove detail
Decomposition = break into parts

🧩 Design

  • Structure diagrams
  • Flowcharts
  • Pseudocode

💻 Coding & Testing

Programs are written and tested repeatedly to remove errors.

🧠 Computer Systems

A system includes hardware, software, data, communication and people.

Systems can be divided into sub-systems.

🔄 Input – Process – Output

Component Description
Input Data entered
Process Actions performed
Output Information produced
Storage Saved data

📐 Algorithms

A step-by-step solution to a problem. 
Price ← 10
Tax ← Price * 0.12
FinalPrice ← Price + Tax

🔀 Selection

IF Age < 18 THEN
  OUTPUT "Child"
ELSE
  OUTPUT "Adult"
ENDIF
  
CASE OF Day
  1 : OUTPUT "Monday"
  OTHERWISE OUTPUT "Invalid"
ENDCASE

🔁 Iteration

FOR i ← 1 TO 10
  OUTPUT "*"
NEXT
  
REPEAT
  INPUT Mark
UNTIL Mark = -1
  
WHILE Mark <> -1 DO
  INPUT Mark
ENDWHILE
