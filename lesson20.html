Lesson 20 – Arrays

Lesson Objectives

  • Understand the concept of arrays
  • Learn how arrays store multiple values
  • Access elements of an array using an index

What is an Array?

An array is a data structure used to store multiple values under a single variable name. Each value in the array is stored in a specific position called an index.

Arrays are useful when a program needs to store a large number of related values.

Example

Suppose we want to store the marks of five students. Instead of creating five separate variables, we can use an array. 

marks[1] ← 75
marks[2] ← 80
marks[3] ← 65
marks[4] ← 90
marks[5] ← 70

Here, marks is the name of the array and each element is accessed using an index number.

Array Structure

An array can be visualised as a list of values stored in memory. 

Index :  1   2   3   4   5
Marks : 75  80  65  90  70

Accessing Array Elements

Individual elements in an array are accessed using their index number.

Example:

OUTPUT marks[3]

This statement will display the value stored at index position 3.

Using Loops with Arrays

Arrays are often used together with loops to process multiple values.

Example algorithm:

FOR i ← 1 TO 5
    INPUT marks[i]
NEXT i

This algorithm stores five values in the array.

Example – Calculating Total Marks

total ← 0

FOR i ← 1 TO 5
    total ← total + marks[i]
NEXT i

OUTPUT total

The loop adds all values stored in the array.

Advantages of Arrays

  • Store many values using one variable name
  • Organise data efficiently
  • Easy to process using loops

Practice Questions

  2. What is an array index?
  3. Write pseudocode to store five numbers in an array.

Exam Style Question

An array called scores stores the marks of 10 students. Write pseudocode using a loop to input the marks into the array.
