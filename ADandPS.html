💡 What is an Algorithm?

Algorithm:
A step-by-step sequence of instructions used to solve a problem or complete a task.

Algorithms are the foundation of all computer programs. Before any code is written, an algorithm must be designed to clearly describe the solution.

🔄 Program Development Life Cycle (PDLC)

The Program Development Life Cycle is a structured process used to develop computer programs. It ensures programs are well planned, tested, and meet user requirements.

Stages of the PDLC

  1. Analysis
  2. Design
  3. Coding
  4. Testing
  5. Maintenance
Exam Tip:
You must be able to describe what happens in each stage, not just list them.

📊 Analysis Stage

Analysis:
The stage where the problem is clearly defined and the requirements of the program are identified.

The output of the analysis stage is the requirements specification. This document describes exactly what the program must do.

Requirements Specification Includes:

  • Inputs required
  • Processing needed
  • Outputs expected
  • Rules and constraints

🧠 Abstraction

Abstraction:
Removing unnecessary details and keeping only the essential information.

Abstraction simplifies complex problems by focusing only on what is relevant.

Example:

A road map shows roads and junctions but does not show buildings, trees, or people.

🧩 Decomposition

Decomposition:
Breaking a complex problem into smaller, manageable parts.

Example – Getting Dressed:

  • Select clothes
  • Remove clothes currently worn
  • Put on selected clothes in the correct order
Exam Warning:
Abstraction removes detail.
Decomposition breaks a problem into parts.
Do not confuse these.
