Question 1 – Car Seat Materials & Colours

a (i) Number of records
Answer: 18 records.
Explanation: Each row represents one combination of code + material + colours.
a (ii) Number of fields
Answer: 9 fields.
Code, Seat Material, Seat Colour, plus 6 paint colours.
a (iii) Primary key suitability
Code is suitable because it is unique.
b SQL Output 
SELECT Code
FROM CAR
WHERE SeatColour = 'Red' AND ('White' OR 'silver');
Condition ('White' OR 'silver') is always TRUE.
So it filters only SeatColour = Red.
c SQL completion 
SELECT *
FROM CAR
WHERE SeatMaterial = 'Leather'
AND (PaintColour = 'Silver' OR PaintColour = 'Grey');

Question 2 – Bicycle Stock Database

a (i) Suitable data types
  • Model number – VARCHAR
  • Description – VARCHAR
  • Colour – VARCHAR
  • Wheel size – INTEGER
  • Price – REAL
  • Manufactured – BOOLEAN
  • Stock – INTEGER
a (ii) Validation
  • Model number – format check
  • Price – range check
  • Stock – range check
a (iii) Primary key
Model number (unique).
b SQL commands
  • SELECT – choose fields
  • FROM – choose table
  • WHERE – filter
  • SUM – total
  • ORDER BY – sort
