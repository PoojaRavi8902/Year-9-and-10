Lesson 21 – Array Processing

Lesson Objectives

  • Understand how arrays are processed using loops
  • Learn how to calculate totals and averages from arrays
  • Apply algorithms to process multiple values stored in arrays

What is Array Processing?

Array processing refers to performing operations on multiple elements stored in an array. Since arrays store many values, loops are commonly used to process each element one by one.

Inputting Values into an Array

A loop can be used to store multiple values into an array. 

FOR i ← 1 TO 5
    INPUT marks[i]
NEXT i

This algorithm stores five values entered by the user into the array marks.

Displaying Array Values

A loop can also be used to display the values stored in an array. 

FOR i ← 1 TO 5
    OUTPUT marks[i]
NEXT i

Calculating the Total of Array Values

total ← 0

FOR i ← 1 TO 5
    total ← total + marks[i]
NEXT i

OUTPUT total

The loop adds each value stored in the array to calculate the total.

Calculating the Average

total ← 0

FOR i ← 1 TO 5
    total ← total + marks[i]
NEXT i

average ← total / 5

OUTPUT average

The algorithm calculates the total first and then divides by the number of values to find the average.

Example Problem

Write an algorithm to input five numbers and display their total and average. 

total ← 0

FOR i ← 1 TO 5
    INPUT numbers[i]
    total ← total + numbers[i]
NEXT i

average ← total / 5

OUTPUT total
OUTPUT average

Why Array Processing is Important

  • Allows programs to handle large amounts of data
  • Reduces repetition in algorithms
  • Makes data processing efficient

Practice Questions

  1. What is array processing?
  2. Write pseudocode to display all elements of an array.
  3. Write pseudocode to calculate the total of five numbers stored in an array.

Exam Style Question

An array called scores stores the marks of 8 students. Write pseudocode to calculate and display the average score.
