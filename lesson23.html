Lesson 23 – Linear Search

Lesson Objectives

  • Understand the purpose of search algorithms
  • Learn how the linear search algorithm works
  • Apply linear search to find values in arrays

What is Searching?

Searching is the process of finding a specific value in a collection of data.

In computer programs, searching is often performed on arrays or lists of values.

One of the simplest search methods is called Linear Search.

What is Linear Search?

Linear search is a method of searching where each element of the array is checked one by one until the required value is found.

If the value is found, the algorithm stops. If the value is not found after checking all elements, the search ends.

Example Array

Index :   1   2   3   4   5
Values:  12  45  30  27  50

Suppose we want to find the value 30. The algorithm checks each element until the value is found.

Linear Search Algorithm

INPUT searchValue

found ← FALSE

FOR i ← 1 TO 5

    IF numbers[i] = searchValue THEN
        found ← TRUE
        OUTPUT "Value found at position ", i
    ENDIF

NEXT i

Example

If the user searches for the number 30, the algorithm checks each element:

  • Check index 1 → not found
  • Check index 2 → not found
  • Check index 3 → value found

Advantages of Linear Search

  • Simple to understand and implement
  • Works with unsorted data
  • Useful for small datasets

Disadvantages of Linear Search

  • Slow for large datasets
  • Must check each element one by one

Practice Questions

  2. Explain how linear search works.
  3. Write pseudocode to search for a number in an array.

Exam Style Question
