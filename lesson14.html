Lesson 14 – Count-Controlled Loops (FOR Loops)

Lesson Objectives

  • Understand the concept of count-controlled loops
  • Learn how FOR loops are used in algorithms
  • Write algorithms that repeat instructions a fixed number of times

What is a Count-Controlled Loop?

A count-controlled loop is used when the number of repetitions is known before the loop begins.

The loop runs a specific number of times using a variable called a loop counter.

In pseudocode, this type of loop is usually written using a FOR loop.

General Structure of a FOR Loop

FOR counter ← startValue TO endValue
    statements
NEXT counter

The counter increases automatically each time the loop repeats. The loop stops when the counter reaches the end value.

Example 1 – Display Numbers from 1 to 5

FOR i ← 1 TO 5
    OUTPUT i
NEXT i

The loop will display the numbers 1, 2, 3, 4, and 5.

Example 2 – Display a Message Multiple Times

FOR i ← 1 TO 3
    OUTPUT "Welcome to Computer Science"
NEXT i

The message will be displayed three times.

Example 3 – Calculate the Total of Numbers

total ← 0

FOR i ← 1 TO 5
    total ← total + i
NEXT i

OUTPUT total

This algorithm calculates the sum of the numbers from 1 to 5.

How the Loop Works

  1. The counter variable is assigned the starting value.
  2. The loop instructions are executed.
  3. The counter increases automatically.
  4. The loop continues until the counter reaches the end value.

Advantages of FOR Loops

  • Simple and easy to understand
  • Useful when the number of repetitions is known
  • Reduces repetition of code

Practice Questions

  1. What is a count-controlled loop?
  2. Write pseudocode to display numbers from 1 to 10.
  3. Write a FOR loop that displays "Hello" five times.

Exam Style Question

Write pseudocode using a FOR loop to display the squares of numbers from 1 to 5.
