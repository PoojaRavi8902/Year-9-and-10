Lesson 12 – CASE Structure

Lesson Objectives

  • Understand the purpose of the CASE structure
  • Learn how CASE is used for multiple decisions
  • Compare CASE statements with IF statements

What is the CASE Structure?

The CASE structure is used when a program must choose between several possible actions depending on the value of a variable.

Instead of writing many IF statements, the CASE structure allows multiple conditions to be handled in a clear and organised way.

General Structure

CASE variable OF
    value1 : statements
    value2 : statements
    value3 : statements
    OTHERWISE : statements
ENDCASE

The program checks the value of the variable and executes the corresponding statements.

Example

Algorithm to display the name of a day based on a number. 

INPUT dayNumber

CASE dayNumber OF
    1 : OUTPUT "Monday"
    2 : OUTPUT "Tuesday"
    3 : OUTPUT "Wednesday"
    4 : OUTPUT "Thursday"
    5 : OUTPUT "Friday"
    OTHERWISE : OUTPUT "Invalid day"
ENDCASE

How the CASE Structure Works

  1. The program evaluates the value of the variable.
  2. The program searches for a matching case.
  3. The corresponding instructions are executed.
  4. If no case matches, the OTHERWISE option is executed.

CASE vs IF Statements

CASE Structure IF Structure
Used for multiple fixed values Used for conditions and comparisons
Clear and organised Can become complex with many conditions
Often easier to read More flexible for complex conditions

Advantages of CASE

  • Improves readability of algorithms
  • Reduces the need for many IF statements
  • Makes decision-making structures clearer

Practice Questions

  1. What is the purpose of the CASE structure?
  2. Write pseudocode using CASE to display a message for numbers 1, 2, or 3.
  3. Explain one advantage of using CASE instead of multiple IF statements.

Exam Style Question

Write pseudocode using a CASE structure that inputs a number between 1 and 3 and displays a corresponding colour:

  • 1 – Red
  • 2 – Blue
  • 3 – Green
