Lesson 18 – Data Types

Lesson Objectives

  • Understand what data types are
  • Identify different types of data used in algorithms
  • Understand why data types are important in programming

What is a Data Type?

A data type defines the type of data that a variable can store. Different types of data require different types of storage and processing.

For example, numbers, text, and logical values are stored using different data types.

Common Data Types

The most commonly used data types in algorithms and programming are:

  • Integer
  • Real
  • Boolean
  • Character
  • String

Integer

An integer is a whole number that does not contain decimal points.

Examples:

10
-5
25

Integers are commonly used for counting and indexing.

Real (Floating Point)

A real number is a number that contains a decimal point.

Examples:

3.14
7.5
-0.25

Real numbers are often used in scientific calculations.

Boolean

A Boolean data type can only have two values:

  • TRUE
  • FALSE

Boolean values are commonly used in conditions and logical operations.

Character

A character represents a single symbol such as a letter, number, or special character.

Examples:

'A'
'7'
'#'

String

A string is a sequence of characters used to represent text.

Examples:

"Hello"
"Computer Science"
"IGCSE"

Why Data Types Are Important

  • They determine how data is stored in memory
  • They ensure correct operations are performed on data
  • They help prevent programming errors

Example Algorithm Using Data Types

INPUT name
INPUT age

OUTPUT "Student Name: ", name
OUTPUT "Student Age: ", age

In this example:

  • name is a string
  • age is an integer
