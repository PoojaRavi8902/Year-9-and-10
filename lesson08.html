Lesson 8 – Arithmetic Operations in Algorithms

Lesson Objectives

  • Understand arithmetic operations used in algorithms
  • Learn how calculations are represented in pseudocode
  • Understand the difference between DIV and MOD
  • Apply arithmetic operations when designing algorithms

Arithmetic Operations in Algorithms

Arithmetic operations are used in algorithms to perform mathematical calculations. These operations allow programs to process numerical data and produce meaningful results.

Arithmetic operations are commonly used in calculations such as totals, averages, areas, and other mathematical computations.

Common Arithmetic Operators

Operator Description Example
+ Addition total ← a + b
- Subtraction difference ← a - b
* Multiplication area ← length * width
/ Division average ← total / count

Integer Division (DIV)

DIV is used to perform integer division. It divides one number by another and returns only the whole number part of the result.

Example:

17 DIV 5 = 3

The decimal part is ignored.

Remainder Operator (MOD)

MOD is used to find the remainder of a division operation.

Example:

17 MOD 5 = 2

This means that 17 divided by 5 leaves a remainder of 2.

Example Algorithm

Write an algorithm that calculates the total and average of two numbers. 

INPUT number1
INPUT number2

total ← number1 + number2
average ← total / 2

OUTPUT total
OUTPUT average

Using DIV and MOD in Algorithms

DIV and MOD are often used together when solving number problems.

Example: Extract the last digit of a number.

INPUT number

lastDigit ← number MOD 10

OUTPUT lastDigit

Example: Extract the first two digits of a number. 

INPUT number

result ← number DIV 100

OUTPUT result

Importance of Arithmetic Operations

  • Used in most algorithms that involve calculations
  • Help process numerical data
  • Essential for solving mathematical problems in programming
