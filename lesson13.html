Lesson 13 – Iteration (Loops)

Lesson Objectives

  • Understand the concept of iteration in algorithms
  • Learn why repetition is necessary in programming
  • Identify situations where loops are useful

What is Iteration?

Iteration is a control structure that allows a set of instructions to be repeated multiple times. This repetition continues until a specific condition is met.

Iteration is often referred to as a loop because the program repeatedly executes the same instructions.

Why Are Loops Needed?

In many problems, a program must perform the same task many times. Instead of writing the same instruction repeatedly, loops allow the task to be repeated automatically.

This makes programs shorter, clearer, and easier to maintain.

Example Without Iteration

OUTPUT "Hello"
OUTPUT "Hello"
OUTPUT "Hello"
OUTPUT "Hello"
OUTPUT "Hello"

This approach is inefficient because the same instruction is written multiple times.

Example Using Iteration

FOR i ← 1 TO 5
    OUTPUT "Hello"
NEXT i

The loop automatically repeats the instruction five times.

Common Types of Loops

There are three main types of loops used in algorithms:

  • Count-controlled loops (FOR loop)
  • Condition-controlled loops (WHILE loop)
  • Condition-controlled loops (REPEAT UNTIL loop)

Each type of loop is used for different situations.

Real-Life Example of Iteration

Consider the task of counting students in a classroom. You repeat the same action for each student until all students are counted.

This repeated process is similar to how loops work in algorithms.

Advantages of Iteration

  • Reduces repetition of instructions
  • Makes programs shorter and more efficient
  • Allows tasks to be repeated automatically
  • Improves program structure

Practice Questions

  1. Define the term iteration.
  2. Why are loops important in programming?
  3. Give one example where iteration might be used.

Exam Style Question

Explain why iteration is useful when designing algorithms.
