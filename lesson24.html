Lesson 24 – Counting Algorithms

Lesson Objectives

  • Understand the purpose of counting algorithms
  • Learn how to count values that meet certain conditions
  • Apply loops and conditions to perform counting

What is a Counting Algorithm?

A counting algorithm is used to determine how many values satisfy a particular condition.

It usually involves checking each element in a dataset and increasing a counter when a condition is true.

Basic Counting Structure

count ← 0

FOR i ← 1 TO n

    IF condition THEN
        count ← count + 1
    ENDIF

NEXT i

Each time the condition is satisfied, the counter increases by one.

Example Problem

Count how many numbers in an array are greater than 50. 

count ← 0

FOR i ← 1 TO 5

    IF numbers[i] > 50 THEN
        count ← count + 1
    ENDIF

NEXT i

OUTPUT count

Example Dataset

Numbers: 45  78  62  30  90

Values greater than 50 are:

  • 78
  • 62
  • 90

So the result of the counting algorithm will be: 

count = 3

Steps of a Counting Algorithm

  1. Set the counter to zero.
  2. Check each value in the dataset.
  3. If the condition is true, increase the counter.
  4. Display the final count.

Why Counting Algorithms Are Useful

  • Used in data analysis
  • Help identify how many values meet specific conditions
  • Common in many real-world programming problems

Practice Questions

  1. What is a counting algorithm?
  2. Write pseudocode to count how many numbers in an array are even.
  3. Write pseudocode to count how many marks are greater than 70.

Exam Style Question

An array called scores stores the marks of 8 students. Write pseudocode to count how many students scored more than 60.
