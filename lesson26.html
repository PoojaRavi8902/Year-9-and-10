Lesson 26 – Maximum and Minimum Algorithms

Lesson Objectives

  • Understand how to find the largest value in a dataset
  • Understand how to find the smallest value in a dataset
  • Apply comparison techniques in algorithms

What is a Maximum Algorithm?

A maximum algorithm is used to find the largest value in a set of numbers.

The algorithm compares each value with the current maximum value and updates the maximum when a larger value is found.

Basic Maximum Algorithm

max ← numbers[1]

FOR i ← 2 TO n

    IF numbers[i] > max THEN
        max ← numbers[i]
    ENDIF

NEXT i

OUTPUT max

The variable max always stores the largest value found so far.

Example Dataset

Numbers: 45  78  62  90  33

The largest value is: 

90

Minimum Algorithm

A minimum algorithm works in a similar way but finds the smallest value. 

min ← numbers[1]

FOR i ← 2 TO n

    IF numbers[i] < min THEN
        min ← numbers[i]
    ENDIF

NEXT i

OUTPUT min

Example Dataset

Numbers: 45  78  62  90  33

The smallest value is: 

33

How the Algorithm Works

  1. Start by assuming the first value is the maximum (or minimum).
  2. Compare each remaining value.
  3. Update the variable if a larger (or smaller) value is found.
  4. Display the final result.

Why These Algorithms Are Important

  • Used to analyse data
  • Help identify highest and lowest values
  • Commonly used in real-world applications

Practice Questions

  1. What is a maximum algorithm?
  2. Write pseudocode to find the largest value in an array.
  3. Write pseudocode to find the smallest value in an array.

Exam Style Question

An array called temperatures stores the temperatures recorded during a week. Write pseudocode to find the highest temperature.
