Lesson 7 – Pseudocode Basics

Lesson Objectives

  • Understand what pseudocode is
  • Learn why pseudocode is used in algorithm design
  • Understand common pseudocode statements used in examinations
  • Write simple algorithms using pseudocode

What is Pseudocode?

Pseudocode is a method used to describe algorithms using structured English that resembles programming language statements. It is not written in any specific programming language but follows logical steps that are easy for humans to understand.

Pseudocode allows programmers to design the logic of a program before writing the actual code. It focuses on the problem-solving steps rather than programming syntax.

Why Do Programmers Use Pseudocode?

  • It helps plan the logic of a program
  • It makes algorithms easier to understand
  • It allows programmers to identify errors before coding
  • It can be easily converted into real programming languages

Basic Structure of Pseudocode

A pseudocode algorithm usually contains three main parts:

  • Input
  • Processing
  • Output

Example Algorithm

Algorithm to calculate the area of a rectangle: 

INPUT length
INPUT width

area ← length × width

OUTPUT area

Common Pseudocode Keywords

Several keywords are commonly used in pseudocode to represent operations.

Keyword Purpose
INPUT Used to receive data from the user
OUTPUT Used to display results
IF Used to make decisions
FOR Used for repetition or loops
WHILE Used for conditional loops
REPEAT Used for loops that execute at least once

Assignment Statements

In pseudocode, the symbol ← is commonly used for assignment. It means that the value on the right is stored in the variable on the left.

Example:

total ← mark1 + mark2 + mark3

This means that the value of the expression is stored in the variable total.

Example Problem

Write pseudocode to calculate the average of three numbers. 

INPUT number1
INPUT number2
INPUT number3

sum ← number1 + number2 + number3
average ← sum / 3

OUTPUT average

Advantages of Pseudocode

  • Easy to read and understand
  • Independent of programming languages
  • Helps identify logical errors
  • Provides a clear design before coding

Practice Questions

  1. Define the term pseudocode.
  2. Explain one advantage of using pseudocode when designing algorithms.
  3. Write pseudocode to calculate the perimeter of a rectangle.

Exam Style Question

Write pseudocode to input two numbers and display their sum.
