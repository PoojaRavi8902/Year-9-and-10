Lesson 17 – Variables

Lesson Objectives

  • Understand the concept of variables in algorithms
  • Learn how variables store data
  • Understand how values are assigned to variables

What is a Variable?

A variable is a named storage location used to store data in a program or algorithm. The value stored in a variable can change during the execution of the program.

Variables allow programs to store and manipulate data when performing calculations or processing information.

Example of a Variable

total ← 0

In this example, the variable total is assigned the value 0.

The symbol ← represents assignment, meaning the value on the right is stored in the variable on the left.

Using Variables in Algorithms

Variables are commonly used to store user input, intermediate results, and final results.

Example algorithm:

INPUT number1
INPUT number2

sum ← number1 + number2

OUTPUT sum

In this example:

  • number1 and number2 store input values.
  • sum stores the result of the calculation.

Changing Variable Values

The value of a variable can change during program execution. 

count ← 1
count ← count + 1

After this operation, the value of count becomes 2.

Variable Naming Rules

When naming variables, the following rules should be followed:

  • Variable names should be meaningful.
  • They should not contain spaces.
  • They should start with a letter.
  • They should not use reserved keywords.

Examples of Good Variable Names

  • totalMarks
  • studentAge
  • temperature

Examples of Poor Variable Names

  • x
  • a
  • value1

Meaningful variable names make algorithms easier to understand.

Practice Questions

  1. Define the term variable.
  2. What does the assignment symbol ← mean?
  3. Write an algorithm that stores a number and displays its square.

Exam Style Question

Explain why variables are important in programming.
