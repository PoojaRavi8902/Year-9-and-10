Lesson 11 – Selection (IF Statements)

Lesson Objectives

  • Understand the concept of selection in algorithms
  • Learn how IF statements are used for decision making
  • Write algorithms that make decisions based on conditions

What is Selection?

Selection is a control structure that allows a program to choose between different actions based on a condition.

In many problems, the program must make decisions depending on the values of data. Selection allows the algorithm to perform different steps depending on whether a condition is TRUE or FALSE.

The IF Statement

The IF statement is used to execute a block of instructions only when a specific condition is true.

General structure:

IF condition THEN
    statements
ENDIF

Example

Algorithm to check if a person is an adult. 

INPUT age

IF age >= 18 THEN
    OUTPUT "Adult"
ENDIF

If the condition is TRUE, the output will be displayed. If the condition is FALSE, the program skips the statement.

IF–ELSE Structure

Sometimes we need to perform one action if the condition is TRUE and another action if the condition is FALSE.

Structure:

IF condition THEN
    statements
ELSE
    statements
ENDIF

Example

Algorithm to determine whether a number is even or odd. 

INPUT number

IF number MOD 2 = 0 THEN
    OUTPUT "Even"
ELSE
    OUTPUT "Odd"
ENDIF

Nested IF Statements

A nested IF statement occurs when an IF statement appears inside another IF statement.

Example:

INPUT score

IF score >= 50 THEN
    OUTPUT "Pass"
ELSE
    OUTPUT "Fail"
ENDIF

Importance of Selection

  • Allows programs to make decisions
  • Controls program behaviour based on conditions
  • Essential for solving real-world problems

Practice Questions

  1. Define the term selection.
  2. What is the purpose of an IF statement?
  3. Write pseudocode to check whether a number is positive.

Exam Style Question

Write pseudocode to input a number and display "Pass" if the number is greater than or equal to 40. Otherwise display "Fail".
