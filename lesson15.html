Lesson 15 – Condition-Controlled Loops (WHILE Loops)

Lesson Objectives

  • Understand the concept of condition-controlled loops
  • Learn how WHILE loops work in algorithms
  • Design algorithms that repeat instructions while a condition is true

What is a Condition-Controlled Loop?

A condition-controlled loop repeats a set of instructions while a specific condition remains true.

Unlike count-controlled loops, the number of repetitions is not known before the loop starts. The loop continues until the condition becomes false.

WHILE Loop Structure

WHILE condition DO
    statements
ENDWHILE

The condition is checked before the loop executes. If the condition is TRUE, the statements inside the loop are executed. If the condition is FALSE, the loop stops.

Example 1 – Counting Numbers

i ← 1

WHILE i <= 5 DO
    OUTPUT i
    i ← i + 1
ENDWHILE

This algorithm displays numbers from 1 to 5.

How the Loop Works

  1. The variable i starts with the value 1.
  2. The program checks the condition i ≤ 5.
  3. If the condition is TRUE, the loop executes.
  4. The value of i increases.
  5. The condition is checked again.

Example 2 – Password Checking

INPUT password

WHILE password <> "secret" DO
    OUTPUT "Incorrect password"
    INPUT password
ENDWHILE

OUTPUT "Access granted"

The program continues asking for the password until the correct one is entered.

Advantages of WHILE Loops

  • Useful when the number of repetitions is unknown
  • Allows programs to respond to changing conditions
  • Widely used in interactive programs

Practice Questions

  1. What is a condition-controlled loop?
  2. Write pseudocode using a WHILE loop to display numbers from 1 to 10.
  3. Explain when a WHILE loop should be used.

Exam Style Question

Write pseudocode using a WHILE loop that asks the user to enter numbers until the number 0 is entered.
